# Changelog

All notable changes to the SlideForge MCP server.

## [1.0.0] — 2026-03-31

### Available

- 13 MCP tools: `render_slide`, `generate_slide`, `generate_deck`, `iterate_slide`, `suggest_template`, `search_templates`, `list_templates`, `list_themes`, `get_slide_status`, `list_jobs`, `get_usage`, `get_capabilities`, `submit_feedback`
- 2 MCP prompts: `create_presentation`, `quick_slide`
- OAuth 2.1 authentication (Claude Desktop native)
- API key authentication (all MCP clients)
- 39 built-in consulting templates
- Custom brand themes
- Inline PNG preview in `get_slide_status`
- Draft mode (gpt-5.4-mini, ~12s) and Pro mode (claude-sonnet-4-6, ~30s)
- $3 free trial on signup
